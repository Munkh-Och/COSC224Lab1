Food Kingdom Website - README

Project Overview
"Food Kingdom" is a food-themed website showcasing a variety of dishes. It includes a navigation bar, dynamic image carousel, portfolio gallery, and a footer with social media links.

Project Structure
- index.html: Main HTML file.
- lab1style.css: Custom CSS for styling.
- assets/images/: Contains images used on the site.
- README.txt: This file.

Technologies Used
- HTML: Structure of the webpage.
- CSS: Custom styling.
- Bootstrap 5: For responsive design (navbar, carousel).
- Bootstrap Icons: Social media icons in the footer.
- JavaScript: Carousel functionality.

Features
- Navbar: Links to "Home" and "Portfolio".
- Image Carousel: Displays food images with navigation.
- Portfolio: Gallery of food items.
- Footer: Social media links and copyright info.

Installation
1. Clone or download the repository.
2. Open index.html in a browser.

How to Use
- Navbar: Click "Home" or "Portfolio" to scroll to sections.
- Carousel: Use the previous/next buttons or indicators.
- Portfolio: View food items in the gallery.

Customization
- Change the background image in lab1style.css.
- Add items to the carousel or portfolio by updating index.html.
- Update social media links in the footer.

Known Issues
- Older browsers may not fully support Bootstrap components.
- Ensure image paths are correct in index.html.
